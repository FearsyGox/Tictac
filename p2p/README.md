# P2P Tic Tac Toe

### Compile p2p.cpp

To compile the program (`p2p.cpp`), use the following command:

```bash
g++ -o p2p.out p2p.cpp
```

## Server

To run the server, use the following command:

```bash
./p2p.out
```

## Client

To run the client, use the following command:

```bash
./p2p.out 127.0.0.1
```
